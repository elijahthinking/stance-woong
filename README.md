## 7. 트러블 슈팅

상황: 푸터 모바일형으로 변환시 UI 오류
해결: 팀원들의 코딩 능력 협조로 인해 문제 해결
과정: 데스크탑 화면에서 보여지는 UI 구조는 작성하였으나 모바일 화면에서 새롭게 나타나는 선을 어떻게 할지 몰라 팀원에게 도움을 구했고 해당 문제를 사사 받고 해결
